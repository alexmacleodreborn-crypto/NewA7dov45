# A7DO v48 — Living World Streamlit App

## Run
```bash
pip install streamlit
streamlit run app.py
```

## Features
- Real-time simulation (2 seconds per tick)
- Full living world with BeenFore City map
- Daily schedules: A7DO, Lorraine, China, all NPCs
- All canonical equations: C(t), W(t), I(t), H(t), M(t), HR(t), ATP(t)
- PCSN: Prediction/Choice/Story/Nodes unified loop
- Neural Physics SafeToSpeak gate
- Jump to any life stage
